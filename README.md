## Project

The final project, **Privacy Preserving / Generalization Bound**, studies how privacy mechanisms in federated learning can be analyzed using tools from information theory, statistics, and learning theory.  
The slides focus on federated learning, differential privacy, Gaussian mechanisms, $E_\epsilon$-divergence, contraction coefficients, and their connection to privacy and generalization.
